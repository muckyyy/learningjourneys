CSP Configuration Details

The following CSP directives were added to support YouTube embeds:

  • script-src: Added https://www.youtube.com, https://s.ytimg.com, https://www.google.com, https://www.gstatic.com
  • style-src: Added https://fonts.googleapis.com
  • img-src: Added https://i.ytimg.com, https://yt3.ggpht.com
  • font-src: Added https://fonts.gstatic.com
  • frame-src: Added https://www.youtube.com, https://www.youtube-nocookie.com
  • media-src: Added https://www.youtube.com, https://*.googlevideo.com, blob:
  • X-Frame-Options: Changed from DENY to SAMEORIGIN

Troubleshooting

If YouTube embeds still don't work:

  1. Check browser console for CSP errors (F12 ‚Üí Console)
  2. Verify Apache configuration was updated and reloaded
  3. Test with browser developer tools network tab
  4. Try both youtube.com and youtube-nocookie.com embeds

Common CSP Error Messages:

  • ERR_BLOCKED_BY_CSP - Content blocked by CSP
  • Refused to frame - Frame-src policy issue
  • Refused to load script - Script-src policy issue
